# HabitBreaker AI Bot

Telegram бот для отслеживания и борьбы с вредными привычками.

## Функциональность

- Создание трекера для вредной привычки
- Отслеживание успешных дней и срывов
- Статистика прогресса
- Мотивационные сообщения
